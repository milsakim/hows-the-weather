# how-s-the-weahter

## 개요
-----------------
- 20개 도시의 현재 날씨와 기상예보를 확인할 수 있는 iOS 애플리케이션

<br/>

## 요구사항 분석서
-----------------
## 공통 사항

1. 서비스가 지원하는 언어는 다음과 같습니다.
    - English
    - 한국어

## 첫 번째 화면

1. 서비스가 서버에서 도시들의 현재 날씨 정보를 가져오는 것에 실패한 경우
    - 서비스는 사용자에게 날씨 정보를 받아오는 것에 실패했음을 alert 형식으로 알립니다.

<br/>

2. 서비스가 서버에서 도시들의 현재 날씨 정보를 가져오는 것에 성공한 경우
    1. 사용자는 서비스에서 도시들의 현재 날씨에 대한 정보를 볼 수 있습니다.
        - 서비스가 지원하는 도시 리스트는 다음과 같습니다.
            - 공주, 광주(전라남도), 구미, 군산, 대구, 대전, 목포, 부산, 서산, 서울, 속초, 수원, 순천, 울산, 익산, 전주, 제주시, 천안, 청주, 춘천
        - 사용자가 서비스에서 확인할 수 있는 도시의 현재 날씨 정보는 다음과 같습니다.
            - 도시 이름, 날씨 아이콘, 현재 기온, 현재 습도

    2. 온도 단위 설정
        1. 서비스가 도시의 현재 날씨 정보를 fetch 하고 있는 경우
            - 서비스는 온도 단위 설정 버튼을 비활성화 시킵니다.
        2. 서비스가 도시의 현재 날씨 정보를 fetch 하고 있지 않은 경우
            - 서비스는 온도 단위 설정 버튼을 활성화 시킵니다.
            - 사용자는 서비스에서 온도 단위를 설정할 수 있습니다.
                - 서비스가 제공하는 온도 단위는 다음과 같습니다.
                    - 섭씨
                    - 화씨
            - 서비스는 사용자가 설정한 온도 단위에 따른 도시들의 현재 날씨 정보를 서버로부터 다시 받아옵니다.

    3. 도시 리스트 정렬
        1. 서비스에서 지원하는 모든 도시들의 현재 날씨 정보가 fetch 되지 않은 경우
            - 서비스는 도시 리스트 정렬 버튼을 비활성화 시킵니다.
        2. 서비스에서 지원하는 모든 도시들의 현재 날씨 정보가 fetch 된 경우
            - 서비스는 도시 리스트 정렬 버튼을 활성화 시킵니다.
            - 사용자는 서비스에서 도시 리스트를 정렬할 수 있습니다.
            - 서비스가 제공하는 도시 리스트 정렬 기준은 다음과 같습니다.
                - 도시 이름 기준 오름차순 / 도시 이름 기준 내림차순
                - 도시의 현재 기온 기준 오름차순 / 도시의 현재 기온 기준 내림차순

    4. 도시들의 현재 날씨 다시 받아오기
        1. 공통 사항
            - 사용자는 도시들의 현재 날씨 테이블을 아래로 잡아 당겨 도시들의 현재 날씨를 서버로부터 다시 받아올 수 있습니다.
        2. 서비스가 도시들의 현재 날씨를 받아오는 것에 실패한 경우
            - 서비스는 사용자에게 날씨 정보를 받아오는 것에 실패했음을 alert 형식으로 알립니다.
            - 사용자가 alert의 ‘재시도’ 버튼을 누를 경우, 서시스는
        3. 서비스가 도시들의 현재 날씨를 받아오는 것에 성공한 경우
            - 2-a와 동일합니다.

    5. 특정 도시의 상세 날씨 확인
        1. 서비스가 도시의 현재 날씨 정보를 서버로부터 받아오고 있는 중인 경우
            - 서비스는 날씨 정보를 받아오고 있는 중임을 사용자에게 alert 형식으로 알립니다.
        2. 서비스가 서버와 통신하고 있지 않은 경우
            - 사용자는 선택한 도시의 상세 날씨 화면(이하 두 번째 화면)으로 전환할 수 있습니다.

## 두 번째 화면

1. 사용자는 서비스에서 도시의 상세 날씨 정보를 확인할 수 있습니다.
    - 서비스가 제공하는 상세 날씨는 다음과 같습다.
        - 도시 이름, 날씨 아이콘, 현재 기온, 체감 기온, 현재 습도, 최저 기온, 최고 기온, 기압, 풍속, 날씨 설명

<br/>

2. 사용자는 ‘미래날씨’ row를 선택해 해당 도시의 기상예보 화면(이하 세 번째 화면)으로 전환할 수 있습니다.

<br/>

## 세번째 화면

1. 서비스가 서버에서 도시의 기상예보 데이터를 가져오는 것에 실패한 경우
    - 서비스는 사용자에게 기상예보 데이터를 가져오는 것이 실패했음을 alert 형식으로 알립니다.
    - 서비스는 사용자를 두 번째 화면으로 전환시킵니다.

<br/>

2. 서비스가 서버에서 도시의 기상예보 데이터를 가져오는 것에 성공한 경우
    - 사용자는 서비스에서 도시의 3시간 단위의 5일 간의 기상예보를 꺾은 선 그래프로 확인할 수 있습니다.
        - 기상예보 그래프의 가로축은 시간을 의미합니다.
        - 기상예보 그래프의 세로축은 다음과 같은 정보를 제공합니다.
            - 최저기온, 최고기온, 습도
    - 서비스는 사용자의 기기 회전에 대응하여 기상예보 그래프를 디스플레이합니다.
    - 서비스는 사용자 기기의 dark mode, light mode에 대응하여 기상예보 그래프를 디스플레이합니다.

<br/>
